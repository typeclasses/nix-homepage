NixOS: The Purely Functional Linux Distribution

Nix is a build system that excels at reproducible builds.

NixOS is a Linux distribution built with Nix.

With NixOS you can build your entire system from a single config file, trivially roll back to previous builds to revert changes, and provision servers and desktops alike with ease.

Get started

Just the package manager

Get reproducible builds and development environments, without leaving your current OS. You can install Nix on and Linux or macOS system.

You need root access upfront to install the package manager, but once it's installed you can install packages without root.

Get the package manager

Make NixOS your distro

If you like Nix, go all the way. Desktops and servers, Nix can build it all.

You can put a NixOS installer onto a USB drive, launch a VM image on services like EC2 and Azure, or start with an existing Linux distro and overwrite it with NixOS.
